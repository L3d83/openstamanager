HTMLBuilder

class HTMLBuilder

Classe dedicata alla gestione della conversione di tag in codice HTML.

Campo di input generico: {[ "type": "text", "required": 1, "value": "$idintervento$" ]}

Campo di testo normale e non modificabile: {[ "type": "span", "value": "$testo$" ]}

Campo select automatizzatp: {[ "type": "select", "required": 1, "values": "query=SELECT id, descrizione FROM co_contratti WHERE idanagrafica=$idanagrafica$", "value": "$idcontratto$" ]}

La sostituzione dei parametri compresi tra $$ viene effettuata attraverso il parametro $records.
